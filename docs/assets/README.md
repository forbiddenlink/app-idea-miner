## Screenshot Guidelines

- **Resolution:** 1920x1080 or higher
- **Format:** PNG with compression
- **Browser:** Chrome or Firefox with dark theme
- **Content:** Use actual data from sample_posts.json
- **Annotations:** Optional - highlight key features

## Taking Screenshots

### macOS
```bash
# Full window
Cmd + Shift + 4, then Space, then click window

# Selected area
Cmd + Shift + 4, then drag
```

### Windows
```bash
# Full window
Alt + PrtScn

# Selected area
Win + Shift + S
```

### Browser Tools
Use browser dev tools (F12) → Device Toolbar → Set to 1920x1080 → Capture
